# 오늘의 기술 핫뉴스 브리핑

**날짜:** 2026-06-06 (토)

기준: 2026-06-05 ~ 06-06 주요 빅테크/AI 소식

---

## 1. OpenAI, GPT-5.3-Codex 공개 — 역대 최강 에이전트 코딩 모델
Codex와 GPT-5 학습 스택을 합쳐 코드 생성·추론·범용 지능을 통합. 기존보다 약 25% 빠르고 주요 벤치마크 신기록 경신. GPT-5.5(Pro/Instant)도 함께 라인업 확장.

## 2. 마이크로소프트, 자체 코딩 모델 MAI-Code-1-Flash 발표
설명을 입력하면 앱·웹사이트 소스코드를 생성하는 MS 첫 코딩 모델. 맥킨지 요구사항에 맞춰 튜닝한 결과 GPT-5.5 대비 성능 우위 + 비용 효율 10배라고 주장. OpenAI 의존도 낮추기 행보.

## 3. 구글, Gemini 3.5 Pro 6월 출시 예고
지난달 I/O에서 GA된 Gemini 3.5 Flash(검색 AI Mode·앱 기본값)에 이어, 추론 격차를 메울 Pro 티어를 6월 중 공개 예정.

## 4. 엔비디아, Computex에서 RTX Spark 슈퍼칩 공개 + Vera CPU 양산
젠슨 황이 6/1 컴퓨텍스 키노트에서 RTX Spark 발표 — 올가을 MS·델·HP·ASUS·레노버·MSI 윈도우 PC에 탑재. CPU·GPU가 메모리를 공유하는 unified memory로 더 큰 AI 모델 구동. 데이터센터용 Vera CPU도 풀 양산(초기 고객: Anthropic, OpenAI, xAI 등).

## 5. Anthropic, Opus 4.8 기본 모델 전환 + Project Glasswing 확대
Opus 4.8이 Max·Team Premium·Enterprise·API 기본값으로. Project Glasswing을 약 150개 신규 조직으로 확대하고, 코드베이스 스캔·패치 제안 기능인 Claude Security 추가.

## 6. GitHub 트렌딩 — OpenClaw, 21만 스타 돌파
로컬 디바이스에서 돌아가는 개인 AI 비서 OpenClaw가 1월 바이럴 이후 폭발 성장, 21만+ 스타 돌파. WhatsApp·Telegram·Slack·Discord·Signal·iMessage 등 50+ 연동을 잇는 로컬 게이트웨이.

## 7. 오픈웨이트 진영 가열 — Kimi K2 Thinking · GLM-5
Kimi K2 Thinking은 200~300회 연속 툴 콜에도 일관성 유지, 복잡한 에이전트 워크플로에서 Claude 4.5 Sonnet급이라는 커뮤니티 평가. GLM-5는 MIT 라이선스 + 코딩 환각 56% 감소. DeepSeek v3.2, Qwen 3.5 등도 프런티어 경쟁 가세.
