# 오늘의 기술 핫뉴스 브리핑

**날짜:** 2026-06-10 (화)

어제~오늘 기준 빅테크·AI 모델·개발자 커뮤니티에서 의미있는 소식만 추렸습니다.

## 1. Apple WWDC 2026 — Gemini로 다시 태어난 Siri
Apple이 6/8 WWDC에서 Google과 손잡고 약 **1.2조 파라미터 맞춤 Gemini 모델**로 재설계한 새 Siri를 공개. 온디바이스 + Private Cloud Compute 조합이며, Google에 연 약 10억 달러를 지불하는 것으로 알려짐. iOS 27, macOS "Golden Gate"도 함께 발표(앱 실행 최대 30%·사진 미리보기 70% 빨라짐).
- 출처: https://www.apple.com/newsroom/2026/06/apple-unveils-next-generation-of-apple-intelligence-siri-ai-and-more/
- 참고: 새 Siri는 EU 규제(상호운용성·프라이버시) 미충족으로 EU에서는 출시하지 않기로 함.

## 2. Meta, 첫 플래그십 LLM "Muse Spark" 공개
Alexandr Wang이 이끄는 Superintelligence Labs에서 만든 Meta의 첫 플래그십 모델. 멀티모달 인식·추론·에이전트 작업에서 경쟁력 있는 성능. 동시에 2026년 AI 설비투자(CapEx)를 **1,150~1,350억 달러**로 책정(작년의 약 2배).
- 출처: https://techstartups.com/2026/06/09/top-tech-news-today-june-9-2026/

## 3. Microsoft, 자체 모델 7종 한꺼번에 출시
OpenAI 의존도를 낮추기 위한 in-house 모델 패밀리(MAI). 플래그십 추론 모델 **MAI-Thinking-1**, GitHub Copilot/VS Code에 깊게 통합된 효율형 코딩 모델 **MAI-Code-1-Flash** 등.
- 출처: https://www.cnbc.com/2026/06/02/microsoft-unveils-new-ai-models-lessen-reliance-on-openai-lower-costs.html
